Folder Structure

When creating the folder structure for the projects their must be a master folder to contain all of the sub folders

When the contract number is unknown the structure must be as follows

XXX - (Ship Name)

If the contract number is known then it must be applied to the folder as followed.

(Contract Number) - (Ship Name)

Here are some examples Below

1114 - Volendam

XXX - Nieuw Amsterdam

1206 - Discovery Princess
